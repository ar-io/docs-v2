# How to Access Data on Arweave

Arweave provides multiple ways to access and retrieve data stored on its permanent storage network. Whether you're looking for specific files, querying large datasets, or building applications that need reliable data access, there are several approaches to choose from based on your specific needs.

The most common methods include direct HTTP requests for simple file retrieval, GraphQL queries for complex data exploration, and specialized tools like Parquet and Iceberg for analytical workloads. Each method offers different trade-offs in terms of performance, complexity, and use case suitability.

For developers building applications, understanding these different access patterns is crucial for creating efficient and reliable systems. The choice of access method often depends on factors like data size, query complexity, real-time requirements, and the specific features your application needs.

This section covers all the major data access methods available on Arweave, from basic HTTP requests to advanced querying techniques, helping you choose the right approach for your specific use case.
