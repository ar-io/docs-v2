# What is Arweave?

Arweave is a decentralized storage network designed to store data permanently with a single upfront payment. Unlike traditional cloud storage where you pay monthly fees and risk data loss, Arweave uses a novel economic model called "permaweb" that ensures your data remains accessible forever. The network is built on a blockchain-like structure called the "blockweave" that connects all stored data in a permanent, immutable chain.

The key innovation of Arweave is its "endowment" model. When you upload data, you pay a one-time fee that covers the cost of storing that data for approximately 200 years. This fee is calculated based on the size of your data and current network conditions, but once paid, your data is guaranteed to be stored permanently. The network uses a proof-of-access consensus mechanism where miners are rewarded for storing and serving data, creating economic incentives for long-term data preservation.

Arweave stores data as individual "transactions" on the network, each with a unique transaction ID that serves as a permanent address. These transactions can contain any type of data—from simple text files to complex web applications, images, videos, or even entire websites. The network automatically replicates data across multiple miners to ensure redundancy and availability, making it virtually impossible for your data to be lost or censored.

The network is designed to be simple yet powerful. You can upload data using various methods, from command-line tools to web interfaces, and access it through any AR.IO gateway using the transaction ID. This makes Arweave the perfect foundation for building permanent, decentralized applications that don't rely on traditional hosting infrastructure.
