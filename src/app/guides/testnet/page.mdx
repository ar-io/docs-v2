# Testnet

The ARIO Network Testnet allows developers to test their applications and workflows using ARIO Network features such as [ArNS Names](/arns) before deploying to the mainnet. The ARIO Network Testnet offers a faucet for requesting testnet ARIO tokens (tARIO).

## Faucet

The ARIO Network Testnet Faucet is a service that allows developers to request testnet ARIO tokens (tARIO). It can be accessed in a browser by visiting [ar://faucet](https://faucet.arweave.net).
