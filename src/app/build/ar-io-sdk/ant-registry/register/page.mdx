# register

`register` is a method on the `ANTRegistry` class that registers an ANT with the registry.

`register` requires authentication.

## Examples

<CodeGroup title="register">

        ```typescript {{title: "NodeJS"}}
            const fs = require("fs");
            const { ANTRegistry, ArweaveSigner } = require(('@ar.io/sdk'))

            async function main() {
                const jwk = JSON.parse(fs.readFileSync("KeyFile.json"))
                const antRegistry = ANTRegistry.init({
                    signer: new ArweaveSigner(jwk)
                })

                const results = await antRegistry.register({
                    processId: "ibJwWk9LTNwnSnc5SFnsJMONyWu0LgIGh11elGOpBqY"
                })
            }

            main()
        ```

        ```typescript {{title: "Web"}}
            const { ANTRegistry, ArConnectSigner } = require(('@ar.io/sdk'))

            async function main() {
                const antRegistry = ANTRegistry.init({
                    signer: new ArConnectSigner(window.arweaveWallet)
                })

                const results = await antRegistry.register({
                    processId: "ibJwWk9LTNwnSnc5SFnsJMONyWu0LgIGh11elGOpBqY"
                })
            }

            main()
        ```
