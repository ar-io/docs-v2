# getArNSReturnedName

`getArNSReturnedName` is a method on the `ARIO` class that retrieves the returned name data for the specified returned name.

`getArNSReturnedName` does not require authentication.

## Examples

<CodeGroup title="getArNSReturnedName">

    ```typescript {{ title: 'NodeJS' }}
    const { ARIO } = require('@ar.io/sdk');

    async function main() {
      const ario = ARIO.init();
      const returnedName = await ario.getArNSReturnedName({ name: 'permalink' });

      console.log(returnedName);
    }

    main();
    ```

    ```typescript {{ title: 'Web' }}
    import { ARIO } from '@ar.io/sdk/web';

    async function main() {
      const ario = ARIO.init();
      const returnedName = await ario.getArNSReturnedName({ name: 'permalink' });

      console.log(returnedName);
    }

    main();
    ```

</CodeGroup>
