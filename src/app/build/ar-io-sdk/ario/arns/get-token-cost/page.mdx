# getTokenCost

`getTokenCost` is a method on the `ARIO` class that calculates the cost in mARIO tokens for a specific interaction (such as buying an ArNS record). The cost is calculated based on the interaction type and its parameters.

`getTokenCost` does not require authentication.

## Examples

<CodeGroup title="getTokenCost">

    ```typescript {{ title: 'NodeJS' }}
    const { ARIO } = require('@ar.io/sdk');

    async function main() {
      const ario = ARIO.init();
      const price = await ario
        .getTokenCost({
            intent: 'Buy-Record',
            name: 'ar-io',
            type: 'permabuy',
        })
        .then((p) => new mIOToken(p).toIO()); // convert to IO tokens for readability
      
      console.log('Cost in IO tokens:', price);
    }

    main();
    ```

