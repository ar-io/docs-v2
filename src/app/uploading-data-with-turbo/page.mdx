# Uploading Data with Turbo

Turbo is a modern bundling service that makes uploading data to Arweave faster, more reliable, and easier to use. Instead of dealing with the complexity of direct Arweave transactions, Turbo handles the bundling process and provides a streamlined API that works with multiple payment methods. This makes it the preferred choice for developers building production applications that need reliable data uploads.

The key advantage of Turbo is its simplified payment model. Instead of needing AR tokens to pay for storage, you can use Turbo Credits that can be purchased with fiat currency or various cryptocurrencies including Ethereum, Solana, Polygon, and more. This removes the barrier of acquiring AR tokens and makes the service accessible to a much wider audience. The service also handles all the technical complexity of bundling transactions and ensures your data is properly stored on Arweave.

Turbo provides both SDK and CLI tools for different use cases. The SDK is perfect for integrating uploads into web applications, while the CLI is great for batch uploads and automation. Both tools support uploading single files, entire folders, and even complete websites with automatic manifest generation. The service also includes features like progress tracking, retry logic, and detailed error reporting to make the upload process as smooth as possible.

Getting started with Turbo is straightforward - you just need to create an account, purchase some credits, and start uploading. The service provides detailed documentation and examples for all major programming languages, making it easy to integrate into your existing workflow. Whether you're building a simple file sharing app or a complex decentralized application, Turbo provides the reliability and ease of use you need for production deployments.

## Quick Start Guide

Here are basic examples of uploading data with the Turbo SDK:

<CodeGroup>
```javascript {{ title: 'Simple Data Upload' }}
import { ArweaveSigner, TurboFactory } from '@ardrive/turbo-sdk'

async function uploadWithTurbo() {
// Load your Arweave wallet
const jwk = JSON.parse(fs.readFileSync('./my-wallet.json', 'utf-8'))
const signer = new ArweaveSigner(jwk)

// Create authenticated Turbo client
const turbo = TurboFactory.authenticated({ signer })

try {
// Upload simple data
const result = await turbo.upload({
data: 'Hello, Arweave!',
events: {
onProgress: ({ totalBytes, processedBytes, step }) => {
console.log(`Progress: ${processedBytes}/${totalBytes} (${step})`)
},
},
})

    console.log('Upload successful!', result.id)

} catch (error) {
console.error('Upload failed:', error)
}
}

````

```javascript {{ title: 'File Upload' }}
import { ArweaveSigner, TurboFactory } from '@ardrive/turbo-sdk'

async function uploadFileWithTurbo() {
  const jwk = JSON.parse(fs.readFileSync('./my-wallet.json', 'utf-8'))
  const signer = new ArweaveSigner(jwk)
  const turbo = TurboFactory.authenticated({ signer })

  try {
    // Upload a file
    const result = await turbo.uploadFile({
      fileStreamFactory: () => fs.createReadStream('./my-file.txt'),
      fileSizeFactory: () => fs.statSync('./my-file.txt').size,
      dataItemOpts: {
        tags: [
          { name: 'Content-Type', value: 'text/plain' },
          { name: 'App-Name', value: 'MyApp' },
        ],
      },
    })

    console.log('File upload successful!', result.id)
  } catch (error) {
    console.error('File upload failed:', error)
  }
}
```

</CodeGroup>

Install the SDK with `npm install @ardrive/turbo-sdk` and you're ready to start uploading!
