# Running your own Infrastructure

Running your own infrastructure on the Arweave network gives you complete control over data access, improved performance, and the ability to contribute to network decentralization. Whether you're running a gateway, bundler, or custom extensions, you can build a robust infrastructure setup tailored to your specific needs.

The core components include AR.IO gateways for data serving, bundler extensions for efficient data submission, and various gateway extensions that add specialized functionality. Each component can be run independently or combined to create a comprehensive infrastructure solution that serves your applications and users.

Getting started is straightforward with Docker-based deployments that handle most of the complexity. You can begin with a simple gateway using a single command, then gradually add more components as your needs grow. The modular architecture means you can scale and customize your infrastructure incrementally.

This section covers everything from basic gateway deployment to advanced configuration, helping you build and maintain your own Arweave infrastructure stack.
