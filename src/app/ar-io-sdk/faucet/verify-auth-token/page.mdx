# verifyAuthToken

`verifyAuthToken` is a method on the `faucet` object that verifies if an authentication token is still valid and not expired.

`verifyAuthToken` does not require authentication.

## Examples

<CodeGroup title="verifyAuthToken">

    ```typescript {{ title: 'NodeJS' }}
    const { ARIO } = require('@ar.io/sdk');

    async function main() {
      const ario = ARIO.testnet(); // faucetUrl defaults to 'https://faucet.ario.permaweb.services'

      const result = await ario.faucet.verifyAuthToken(
        'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...'
      );

      console.log('Token valid:', result.valid);
    }

    main();
    ```
