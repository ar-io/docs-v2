# getGateways

`getGateways` is a method on the `ARIO` class that retrieves all gateways registered in the network with support for pagination and filtering.

`getGateways` does not require authentication.

## Examples

<CodeGroup title="getGateways">

    ```typescript {{ title: 'NodeJS' }}
    const { ARIO } = require('@ar.io/sdk');

    async function main() {
      const ario = ARIO.init();
      const gateways = await ario.getGateways({
        limit: 50,
        sortBy: 'operatorStake',
        sortOrder: 'desc'
      });

      console.log(gateways);
    }

    main();
    ```

    ```typescript {{ title: 'Web' }}
    import { ARIO } from '@ar.io/sdk/web';

    async function main() {
      const ario = ARIO.init();
      const gateways = await ario.getGateways({
        limit: 50,
        sortBy: 'operatorStake',
        sortOrder: 'desc'
      });

      console.log(gateways);
    }

    main();
    ```
