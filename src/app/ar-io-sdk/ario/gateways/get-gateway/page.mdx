# getGateway

`getGateway` is a method on the `ARIO` class that retrieves detailed information about a specific gateway using its address.

`getGateway` does not require authentication.

## Examples

<CodeGroup title="getGateway">

    ```typescript {{ title: 'NodeJS' }}
    const { ARIO } = require('@ar.io/sdk');

    async function main() {
      const ario = ARIO.init();
      const gateway = await ario.getGateway({
        address: 't4Xr0_J4Iurt7caNST02cMotaz2FIbWQ4Kbj616RHl3',
      });

      console.log(gateway);
    }

    main();
    ```

