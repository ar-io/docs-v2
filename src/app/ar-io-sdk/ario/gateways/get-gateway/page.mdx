# getGateway

`getGateway` is a method on the `ARIO` class that retrieves a gateway's information using its staking wallet address.

`getGateway` does not require authentication.

## Examples

<CodeGroup title="getGateway">

    ```typescript {{ title: 'NodeJS' }}
    const { ARIO } = require('@ar.io/sdk');

    async function main() {
      const ario = ARIO.init();
      const gateway = await ario.getGateway({
        address: 'f_EcJ8tYOFYjBMCIi-dlmzg8AiWZ2hELrXRFWPKCv9I',
      });

      console.log(gateway);
    }

    main();
    ```

