# getArNSRecords

`getArNSRecords` is a method on the `ARIO` class that retrieves all registered ArNS records, with support for pagination and custom sorting. Pagination is handled using a cursor system, where the cursor is the ArNS name from the last record of the previous request.

`getArNSRecords` does not require authentication.

## Example

<CodeGroup title="getArNSRecords">

    ```javascript {{ title: 'NodeJS' }}
    const { ARIO } = require('@ar.io/sdk');

    async function main() {
      const ario = ARIO.init();
      // get the newest 100 names
        const records = await ario.getArNSRecords({
                limit: 100,
                sortBy: 'startTimestamp',
                sortOrder: 'desc',
        });

      console.log(records);
    }

    main();
    ```

