# getTokenCost

`getTokenCost` is a method on the `ARIO` class that calculates the cost in mARIO tokens for various ArNS operations such as buying records, extending leases, increasing undername limits, upgrading to permabuy, and requesting primary names.

`getTokenCost` does not require authentication.

## Parameters

The `getTokenCost` method accepts different parameter sets depending on the `intent` (the specific action you want to check the cost for). Each intent requires a different combination of parameters as outlined below:

## Examples

<CodeGroup title="getTokenCost">

    ```typescript {{ title: 'NodeJS' }}
    const { ARIO } = require('@ar.io/sdk');

    async function main() {
      const ario = ARIO.init();

      // Buy a permabuy name
      const permabuyPrice = await ario.getTokenCost({
        intent: 'Buy-Record',
        name: 'ar-io',
        type: 'permabuy'
      });

      // Buy a 1-year lease
      const leasePrice = await ario.getTokenCost({
        intent: 'Buy-Record',
        name: 'ar-io',
        type: 'lease',
        years: 1
      });

      // Extend lease for 2 years
      const extensionPrice = await ario.getTokenCost({
        intent: 'Extend-Lease',
        name: 'ar-io',
        years: 2
      });

      // Increase undername limit by 10
      const increasePrice = await ario.getTokenCost({
        intent: 'Increase-Undername-Limit',
        name: 'ar-io',
        quantity: 10
      });

      // Upgrade lease to permabuy
      const upgradePrice = await ario.getTokenCost({
        intent: 'Upgrade-Name',
        name: 'ar-io'
      });

      // Request primary name
      const primaryNamePrice = await ario.getTokenCost({
        intent: 'Primary-Name-Request',
        name: 'ar-io'
      });

      console.log({
        permabuyPrice,      // in mARIO
        leasePrice,         // in mARIO
        extensionPrice,     // in mARIO
        increasePrice,      // in mARIO
        upgradePrice,       // in mARIO
        primaryNamePrice    // in mARIO
      });
    }

    main();
    ```
