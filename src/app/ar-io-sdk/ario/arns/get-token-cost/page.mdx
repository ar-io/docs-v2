# getTokenCost

`getTokenCost` is a method on the `ARIO` class that calculates the cost in mARIO tokens for various ArNS operations such as buying records, extending leases, increasing undername limits, and more.

`getTokenCost` does not require authentication.

## Parameters

The method accepts different parameter sets depending on the operation:

## Examples

<CodeGroup title="getTokenCost">

    ```typescript {{ title: 'NodeJS' }}
    const { ARIO } = require('@ar.io/sdk');

    async function main() {
      const ario = ARIO.init();

      // Buy a permabuy name
      const permabuyPrice = await ario.getTokenCost({
        intent: 'Buy-Record',
        name: 'ar-io',
        type: 'permabuy'
      });

      // Buy a 1-year lease
      const leasePrice = await ario.getTokenCost({
        intent: 'Buy-Record',
        name: 'ar-io',
        type: 'lease',
        years: 1
      });

      // Extend lease for 2 years
      const extensionPrice = await ario.getTokenCost({
        intent: 'Extend-Lease',
        name: 'ar-io',
        years: 2
      });

      console.log({
        permabuyPrice,    // in mARIO
        leasePrice,       // in mARIO
        extensionPrice    // in mARIO
      });
    }

    main();
    ```
