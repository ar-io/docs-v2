# getVaults

`getVaults` is a method on the `ARIO` class that retrieves all vaults owned by a specific wallet address.

`getVaults` does not require authentication.

## Example

<CodeGroup title="getVaults">

    ```typescript {{ title: 'NodeJS' }}
    const { ARIO } = require('@ar.io/sdk');

    async function main() {
      const ario = ARIO.init();
      const vaults = await ario.getVaults({
        address: "ccp3blG__gKUvG3hsGC2u06aDmqv4CuhuDJGOIg0jw4"
      });

      console.log(vaults);
    }

    main();
    ```

