# getVault

`getVault` is a method on the `ARIO` class that retrieves detailed information about a specific vault using the owner's address and vault ID.

`getVault` does not require authentication.

## Examples

<CodeGroup title="getVault">

    ```typescript {{ title: 'NodeJS' }}
    const { ARIO } = require('@ar.io/sdk');

    async function main() {
      const ario = ARIO.init();
      const vault = await ario.getVault({
        address: "ccp3blG__gKUvG3hsGC2u06aDmqv4CuhuDJGOIg0jw4",
        vaultId: "vault-123"
      });

      console.log(vault);
    }

    main();
    ```

    ```typescript {{ title: 'Web' }}
    import { ARIO } from '@ar.io/sdk/web';

    }

    main();
    ```
