# getVault

`getVault` is a method on the `ARIO` class that retrieves the locked-balance user vault of the ARIO process by the specified wallet address and vault ID.

`getVault` does not require authentication.

## Examples

<CodeGroup title="getVault">

    ```typescript {{ title: 'NodeJS' }}
    const { ARIO } = require('@ar.io/sdk');

    async function main() {
      const ario = ARIO.init();
      const vault = await ario.getVault({
        address: 'QGWqtJdLLgm2ehFWiiPzMaoFLD50CnGuzZIPEdoDRGQ',
        vaultId: 'vaultIdOne',
      });

      console.log(vault);
    }

    main();
    ```
