# getObservations

`getObservations` is a method on the `ARIO` class that returns the epoch-specific observation list. If no epoch index is provided, the current epoch is used.

`getObservations` does not require authentication.

## Examples

<CodeGroup title="getObservations">

    ```typescript {{ title: 'NodeJS' }}
    const { ARIO } = require('@ar.io/sdk');

    async function main() {
      const ario = ARIO.init();
      const observations = await ario.getObservations();

      console.log(observations);
    }

    main();
    ```

    ```typescript {{ title: 'Web' }}
    import { ARIO } from '@ar.io/sdk/web';

    async function main() {
      const ario = ARIO.init();
      const observations = await ario.getObservations();

      console.log(observations);
    }

    main();
    ```

</CodeGroup>
