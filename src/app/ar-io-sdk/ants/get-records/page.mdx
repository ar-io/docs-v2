# getRecords

`getRecords` is a method on the `ANT` class that retrieves all the records stored in the ANT process, including both base name records and undername records.

`getRecords` does not require authentication.

## Examples

<CodeGroup title="getRecords">

    ```typescript {{title: "Basic"}}
    const records = await ant.getRecords();
    ```

    ```typescript {{ title: 'NodeJS' }}
    const { ANT } = require('@ar.io/sdk');

    async function main() {
      const ant = ANT.init({ processId: "bh9l1cy0aksiL_x9M359faGzM_yjralacHIUo8_nQXM" });
      const records = await ant.getRecords();

      console.log(records);
    }

    main();
    ```

    ```typescript {{ title: 'Web' }}
    import { ANT } from '@ar.io/sdk/web';

    async function main() {
      const ant = ANT.init({ processId: "bh9l1cy0aksiL_x9M359faGzM_yjralacHIUo8_nQXM" });
      const records = await ant.getRecords();

      console.log(records);
    }

    main();
    ```

</CodeGroup>

## Parameters

The `getRecords` method does not accept any parameters.
