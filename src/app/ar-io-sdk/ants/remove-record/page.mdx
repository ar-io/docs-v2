# removeRecord

`removeRecord` is a method on the `ANT` class that removes a specified record from the ANT process.

`removeRecord` requires authentication.

## Examples

<CodeGroup title="removeRecord">

       ```typescript {{title: "NodeJS"}}
        const fs = require("fs");
        const { ANT, ArweaveSigner } = require("@ar.io/sdk");

        async function main() {
        const jwk = JSON.parse(fs.readFileSync("KeyFile.json"));
        const ant = ANT.init({
            signer: new ArweaveSigner(jwk),
            processId: "bh9l1cy0aksiL_x9M359faGzM_yjralacHIUo8_nQXM"
        });

        const { id: txId } = await ant.removeRecord(
                {
                    undername: 'remove-domain',
                },
                // optional additional tags
                { tags: [{ name: 'App-Name', value: 'My-Awesome-App' }] },
            );
        }

        main();

    ```
