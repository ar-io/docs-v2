# ANT Class

## Overview

The `ANT` class is a core component of the AR.IO SDK that provides a comprehensive interface for interacting with Arweave Name Token (ANT) contracts. ANTs are smart contracts that control ArNS names and provide functionality for managing records, controllers, and metadata.

The ANT class supports both read and write operations, allowing developers to:

- Retrieve ANT state and information
- Manage ANT configuration and metadata
- Control ArNS name records and undernames
- Handle token balances and transfers
- Manage controllers and permissions

## Initialization

The ANT class can be initialized in different ways depending on your needs:

```typescript
import { ANT } from '@ar.io/sdk'

// Read-only mode (no signer required)
const ant = ANT.init({
  processId: 'your-ant-process-id',
})

// Read-write mode (signer required for write operations)
const ant = ANT.init({
  processId: 'your-ant-process-id',
  signer: new ArweaveSigner(jwk),
})
```

## Examples

### Basic Read Operations

```typescript
import { ANT } from '@ar.io/sdk'

const ant = ANT.init({
  processId: 'your-ant-process-id',
})

// Get basic information
const info = await ant.getInfo()
console.log('ANT Name:', info.name)
console.log('ANT Ticker:', info.ticker)

// Get all records
const records = await ant.getRecords()
console.log('Total Records:', records.length)
```

### Write Operations

```typescript
import { ANT, ArweaveSigner } from '@ar.io/sdk'

const ant = ANT.init({
  processId: 'your-ant-process-id',
  signer: new ArweaveSigner(jwk),
})

// Set a new record
const { id: txId } = await ant.setRecord({
  name: 'example',
  transactionId: 'your-tx-id',
  ttlSeconds: 3600,
})

console.log('Record set successfully:', txId)
```

## Notes

- Read operations do not require authentication and can be performed with just a process ID
- Write operations require a signer to be provided during initialization
- The ANT class automatically handles AO process communication and state management
- All write operations return a transaction ID that can be used to track the operation
