# accessControlList

`accessControlList` is a method on the `ANTRegistry` class that retrieves the access control list (ACL) for the ANT Registry, showing which addresses have permission to perform various operations.

`accessControlList` does not require authentication.

## Parameters

The `accessControlList` method does not accept any parameters.

## Examples

<CodeGroup title="accessControlList">

    ```typescript {{ title: 'NodeJS' }}
    const { ANTRegistry } = require('@ar.io/sdk');

    async function main() {
      const antRegistry = ANTRegistry.init();
      const acl = await antRegistry.accessControlList();

      console.log(acl);
    }

    main();
    ```

    ```typescript {{ title: 'Web' }}
    import { ANTRegistry } from '@ar.io/sdk/web';

    async function main() {
      const antRegistry = ANTRegistry.init();
      const acl = await antRegistry.accessControlList();

      console.log(acl);
    }

    main();
    ```

</CodeGroup>
