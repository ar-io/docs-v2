# accessControlList

`accessControlList` is a method on the `ANTRegistry` class that receives a list of ANTs owned or controlled by the provided wallet address.

`accessControlList` does not require authentication

## Parameters

## Examples

<CodeGroup title="accessControlList">

        ```typescript {{title: "NodeJS"}}
            const { ANTRegistry } = require('@ar.io/sdk')

            async function main() {
                const antRegistry = ANTRegistry.init()

                const results = await antRegistry.accessControlList({
                    address: "GtDQcrr2QRdoZ-lKto_S_SpzEwiZiHVaj3x4jAgRh4o"
                })

                console.log(results)
            }

            main()
        ```
