# addVersion

`addVersion` is a method on the `ANTVersions` class that adds a new version to an ANT. This method requires authentication and a signer to be provided during initialization.

## Examples

<CodeGroup title="addVersion">

    ```typescript {{title: "Basic"}}
    const { id: txId } = await versions.addVersion({
        version: "2.0.0",
        moduleId: "abc123...def789",
        luaSourceId: "ghi012...jkl345",
        notes: "Major update with new features"
    });
    ```

    ```typescript {{ title: 'NodeJS' }}
    const { ANTVersions, ArweaveSigner } = require('@ar.io/sdk');

    async function main() {
      const jwk = JSON.parse(fs.readFileSync("KeyFile.json"));
      const versions = ANTVersions.init({
        processId: "bh9l1cy0aksiL_x9M359faGzM_yjralacHIUo8_nQXM",
        signer: new ArweaveSigner(jwk)
      });

      const { id: txId } = await versions.addVersion({
        version: "2.0.0",
        moduleId: "abc123...def789",
        luaSourceId: "ghi012...jkl345",
        notes: "Major update with new features"
      });

      console.log(txId);
    }

    main();
    ```

    ```typescript {{ title: 'Web' }}
    import { ANTVersions, ArConnectSigner } from '@ar.io/sdk/web';

    async function main() {

      console.log(txId);
    }

    main();
    ```
