## How to Get Started Running Your Own AR.IO Gateway

**Prerequisites:** [Docker installed](https://docs.docker.com/engine/install/) on your system

**Run a gateway in 30 seconds:**

```bash
docker run -p 4000:4000 -v ar-io-data:/app/data ghcr.io/ar-io/ar-io-core:latest
```

**Test your gateway:**

```bash
curl localhost:4000/agYcCFJtrMG6cqMuZfskIkFTGvUPddICmtQSBIoPdiA
# Should return: 1984
```
