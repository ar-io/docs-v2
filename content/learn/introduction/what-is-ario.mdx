# What is AR.IO?

AR.IO is the world's first **permanent cloud network**.

## The Foundation

Built on Arweave, AR.IO creates a global ecosystem of gateways and services that connect users to the permaweb.

## The Network

AR.IO is an open, distributed, and ownerless system supported by operators worldwide.

## The Gateways

Decentralized nodes act as "Permanent Cloud Service Providers" delivering critical services.

## The Services

- Read, write, index and query data
- Permanent domain name system
- Location-independent access

## The Token

Powered by ARIO token for network functionality and accessibility.

## The Goal

A permanent cloud network designed to preserve data and expand web possibilities.
