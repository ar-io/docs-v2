# Welcome to the Permaweb

Data in paradise. The AR.IO ecosystem is dedicated to cultivating products and protocols for sustaining access to digital permanence, making the permaweb available to everyone. Powered by the ARIO Token, this global network of Gateways connects users to permanently stored data, files, applications, and web pages on the Arweave decentralized storage network.

## Getting Started

<Tabs items={["What is AR.IO?", "Key Concepts", "Start Building"]}>
  <Tab value="What is AR.IO?">
    ### The AR.IO Network AR.IO is a decentralized gateway network that provides
    access to data stored permanently on Arweave. The network consists of: -
    **Gateways**: Nodes that cache, serve, and index Arweave data - **ArNS**:
    Human-readable names for permanent data (like DNS for Arweave) - **ARIO
    Token**: The native token powering network incentives - **Permaweb**: The
    permanent web built on top of Arweave storage ### Why AR.IO? - **Permanent
    Access**: Data stored once is accessible forever - **Decentralized**: No
    single point of failure or censorship - **Developer Friendly**: SDKs and
    APIs for easy integration - **Economic Incentives**: Earn rewards by running
    gateway infrastructure
  </Tab>

<Tab value="Key Concepts">
  ### Core Technologies **Arweave** The underlying blockchain that provides
  permanent data storage through its unique blockweave structure and endowment
  mechanism. **Gateways** AR.IO gateways are the access layer for Arweave data,
  providing caching, indexing, and data retrieval services. **ArNS (Arweave Name
  System)** A decentralized naming system that maps human-readable names to
  Arweave transaction IDs, similar to DNS. **Wayfinder** A protocol for
  discovering and connecting to the best available gateways based on performance
  and reliability. **Observer-Reported Incentive Protocol (OIP)** The system
  that tracks gateway performance and distributes rewards to high-performing
  gateway operators.
</Tab>

  <Tab value="Start Building">

    names](/learn/arns) for your content - Access the permaweb through
    [ar.io](https://ar.io)
  </Tab>
</Tabs>
