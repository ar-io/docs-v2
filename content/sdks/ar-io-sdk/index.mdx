This is the home of [ar.io] SDK. This SDK provides functionality for interacting with the AR.IO Network contract, buying ArNS names, managing ANTs, and more. It is available for both NodeJS and Web environments.

## Quick Start

<Tabs items={['Node.js', 'Web']}>
  <Tab value="Node.js">
    <Steps>
      <Step>
        ### Install the SDK

        ```npm
        npm install @ar.io/sdk
        ```
      </Step>
      <Callout type="info">
      The `--ignore-engines` flag is required when using yarn, as [permaweb/aoconnect](http://github.com/permaweb/ao) recommends only the use of npm. Alternatively, you can add a `.yarnrc.yml` file to your project containing `ignore-engines true` to ignore the engines check.
      </Callout>
      <Step>
        ### Import and configure

        ```javascript
        import { ARIO } from '@ar.io/sdk/node';

        // set up client
        const ario = ARIO.mainnet();
        ```
      </Step>
      <Step>
        ### Make your first request

        ```javascript
        // fetch gateways
        const gateways = await ario.getGateways();
        console.log(gateways);
        ```
      </Step>
    </Steps>

  </Tab>
  
  <Tab value="Web">
    <Steps>
      <Step>
        ### Install the SDK

        ```npm
        npm install @ar.io/sdk
        ```
      </Step>
      <Step>
        ### Install polyfills

        <Callout type="warning" title="Browser Polyfills Required">
        When using ar-io-sdk in browser environments, you need to configure polyfills. For example, with Vite:
        </Callout>

        ```npm
        npm install --save-dev vite-plugin-node-polyfills
        ```

        ```js
        // vite.config.js
        import { defineConfig } from 'vite';
        import { nodePolyfills } from 'vite-plugin-node-polyfills';

        export default defineConfig({
          plugins: [
            nodePolyfills({
              globals: {
                Buffer: true,
                global: true,
                process: true,
              },
            }),
          ],
        });
        ```
        
      </Step>
      <Step>
        ### Import and configure

        ```javascript
        import { ARIO } from '@ar.io/sdk/web';

        // set up client
        const ario = ARIO.mainnet();
        ```
      </Step>
      <Step>
        ### Make your first request

        ```javascript
        // fetch gateways
        const gateways = await ario.getGateways();
        console.log(gateways);
        ```
      </Step>
      <Step>
        ### Use wallet for signed requests

        For operations that require signatures (like purchasing ArNS names), integrate with a wallet:

        ```javascript
        import { ARIO, createAoSigner } from '@ar.io/sdk/web';

        // After connecting with a wallet (e.g., ArConnect or Wander)
        const signer = createAoSigner(window.arweaveWallet);
        
        // Create authenticated client
        const authenticatedArio = ARIO.mainnet({ signer });
        
        // Now you can perform signed operations
        const record = await authenticatedArio.buyRecord({
          name: 'my-arns-name',
          type: 'lease',
          years: 1
        });
        ```

        <Callout type="info">
          We recommend using [Wander](https://docs.wander.app/wander-connect/intro) for seamless wallet integration.
        </Callout>
      </Step>
    </Steps>

  </Tab>
</Tabs>
