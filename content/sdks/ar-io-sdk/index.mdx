This is the home of [ar.io] SDK. This SDK provides functionality for interacting with the AR.IO Network contract, buying ArNS names, managing ANTs, and more. It is available for both NodeJS and Web environments.

## Quick Start

<Tabs items={['Node.js', 'Web']}>
  <Tab value="Node.js">
    <Steps>
      <Step>
        ### Install the SDK

        ```bash
        npm install @ar.io/sdk
        // or
        yarn add @ar.io/sdk --ignore-engines
        ```
      </Step>
      <Callout type="info">
      The `--ignore-engines` flag is required when using yarn, as [permaweb/aoconnect] recommends only the use of npm. Alternatively, you can add a `.yarnrc.yml` file to your project containing `ignore-engines true` to ignore the engines check.
      </Callout>
      <Step>
        ### Import and configure

        ```javascript
        import { ARIO } from '@ar.io/sdk/node';

        // set up client
        const ario = ARIO.mainnet();
        ```
      </Step>
      <Step>
        ### Make your first request

        ```javascript
        // fetch gateways
        const gateways = await ario.getGateways();
        console.log(gateways);
        ```
      </Step>
    </Steps>

  </Tab>
  
  <Tab value="Web">
    <Steps>
      <Step>
        ### Install the SDK

        ```bash
        npm install @ar.io/sdk
        ```
      </Step>
      <Step>
        ### Install polyfills

        <Callout type="warning" title="Browser Polyfills Required">
        When using ar-io-sdk in browser environments, you need to configure polyfills. For example, with Vite:
        
        ```bash
        npm install --save-dev vite-plugin-node-polyfills
        ```
        
        See the [installation notes](/sdks/wayfinder/wayfinder-core/installation-notes) for detailed configuration examples.
        </Callout>
      </Step>
      <Step>
        ### Configure your bundler

        Configure your bundler (Webpack, Vite, Rollup, etc.) to provide polyfills for `crypto`, `process`, and `buffer`.

        Refer to the [examples directory](https://github.com/ar-io/ar-io-sdk/tree/main/examples) for Webpack and Vite configurations.
      </Step>
      <Step>
        ### Import and configure

        ```javascript
        import { ARIO } from '@ar.io/sdk/web';

        // set up client
        const ario = ARIO.mainnet();
        ```
      </Step>
      <Step>
        ### Make your first request

        ```javascript
        // fetch gateways
        const gateways = await ario.getGateways();
        console.log(gateways);
        ```
      </Step>
    </Steps>

  </Tab>
</Tabs>
