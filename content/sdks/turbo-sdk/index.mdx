# Turbo SDK

SDK for interacting with Turbo, a fast and efficient data upload service for Arweave. Turbo provides high-performance uploads with instant confirmation and transparent pricing.

## Quick Start

<Tabs items={['Node.js', 'Web']}>
  <Tab value="Node.js">
    <Steps>
      <Step>
        ### Install the SDK

        ```bash
        npm install @ardrive/turbo-sdk
        ```
      </Step>
      <Step>
        ### Import and configure

        ```javascript
        import { TurboFactory } from '@ardrive/turbo-sdk';

        // Create a Turbo client
        const turbo = TurboFactory.authenticated({
          privateKey: 'your-private-key'
        });
        ```
      </Step>
      <Step>
        ### Upload data

        ```javascript
        // Upload data to Arweave via Turbo
        const uploadResult = await turbo.uploadFile({
          fileStreamFactory: () => fs.createReadStream('path/to/file'),
          fileSizeFactory: () => fs.statSync('path/to/file').size,
          signal: AbortSignal.timeout(60_000),
        });

        console.log('Upload complete:', uploadResult);
        ```
      </Step>
    </Steps>

  </Tab>
  
  <Tab value="Web">
    <Steps>
      <Step>
        ### Install the SDK

        ```bash
        npm install @ardrive/turbo-sdk
        ```
      </Step>
      <Step>
        ### Install polyfills (required for web environments)

        <Callout type="warn">
        Polyfills are required for web environments when using bundlers.
        </Callout>

        ```bash
        npm install --save-dev crypto-browserify stream-browserify buffer
        ```

        Configure your bundler (Webpack, Vite, Rollup, etc.) to provide polyfills for `crypto`, `process`, and `buffer`. Refer to your bundler's documentation for polyfill configuration.
      </Step>
      <Step>
        ### Import and configure

        ```javascript
        import { TurboFactory } from '@ardrive/turbo-sdk/web';

        // Create a Turbo client
        const turbo = TurboFactory.authenticated({
          privateKey: 'your-private-key'
        });
        ```
      </Step>
      <Step>
        ### Create a file upload component

        ```jsx
        import React, { useState } from 'react';
        import { TurboFactory } from '@ardrive/turbo-sdk/web';

        function FileUploader() {
          const [uploading, setUploading] = useState(false);
          const [result, setResult] = useState(null);

          const handleFileUpload = async (event) => {
            const file = event.target.files[0];
            if (!file) return;

            setUploading(true);
            try {
              const turbo = TurboFactory.authenticated({
                privateKey: 'your-private-key'
              });

              const uploadResult = await turbo.uploadFile({
                fileStreamFactory: () => file.stream(),
                fileSizeFactory: () => file.size,
                signal: AbortSignal.timeout(60_000),
              });

              setResult(uploadResult);
              console.log('Upload complete:', uploadResult);
            } catch (error) {
              console.error('Upload failed:', error);
            } finally {
              setUploading(false);
            }
          };

          return (
            <div>
              <input
                type="file"
                onChange={handleFileUpload}
                disabled={uploading}
              />
              {uploading && <p>Uploading...</p>}
              {result && (
                <div>
                  <p>Upload successful!</p>
                  <p>Transaction ID: {result.id}</p>
                </div>
              )}
            </div>
          );
        }

        export default FileUploader;
        ```
      </Step>
    </Steps>

  </Tab>
</Tabs>
