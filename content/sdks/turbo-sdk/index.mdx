SDK for interacting with Turbo, a fast and efficient data upload service for Arweave. Turbo provides high-performance uploads with instant confirmation and transparent pricing.

## Quick Start

<Tabs items={['Node.js', 'Web (React)', 'Web (HTML)']}>
  <Tab value="Node.js">
    <Steps>
      <Step>
        ### Install the SDK

        ```npm
        npm install @ardrive/turbo-sdk
        ```
      </Step>
      <Step>
        ### Import and configure

        ```javascript
        import { TurboFactory } from '@ardrive/turbo-sdk';

        // Create a Turbo client
        const turbo = TurboFactory.authenticated({
          privateKey: 'your-private-key'
        });
        ```
      </Step>
      <Step>
        ### Upload data

        ```javascript
        // Upload data to Arweave via Turbo
        const uploadResult = await turbo.uploadFile({
          fileStreamFactory: () => fs.createReadStream('path/to/file'),
          fileSizeFactory: () => fs.statSync('path/to/file').size,
          signal: AbortSignal.timeout(60_000),
        });

        console.log('Upload complete:', uploadResult);
        ```
      </Step>
    </Steps>

  <Tab value="Web (React)">
    <Steps>
      <Step>
        ### Install the SDK

        ```npm
        npm install @ardrive/turbo-sdk
        ```
      </Step>
      <Step>
        ### Install polyfills (required for web environments)

        <Callout type="warn">
        Polyfills are required for web environments when using Turbo-SDK in web environments.
        </Callout>

        ```npm
        npm install --save-dev vite-plugin-node-polyfills
        ```

        ```js
        // vite.config.js
        import { defineConfig } from 'vite';
        import { nodePolyfills } from 'vite-plugin-node-polyfills';

        export default defineConfig({
          plugins: [
            nodePolyfills({
              globals: {
                Buffer: true,
                global: true,
                process: true,
              },
            }),
          ],
        });
        ```

        Configure your bundler (Webpack, Vite, Rollup, etc.) to provide polyfills for `crypto`, `process`, and `buffer`. Refer to your bundler's documentation for polyfill configuration.
      </Step>
      <Step>
        ### Connect wallet

        <Callout type="info">
          For production applications, we recommend using [Wander](https://docs.wander.app/wander-connect/intro) for seamless wallet connection.
        </Callout>

        First, set up wallet connection to securely sign transactions:

        ```jsx
        import { useState, useEffect } from 'react';

        function useWallet() {
          const [wallet, setWallet] = useState(null);
          const [address, setAddress] = useState(null);

          const connectWallet = async () => {
            // Check if wallet is available (e.g., ArConnect or Wander)
            if (!window.arweaveWallet) {
              alert('Please install a wallet extension like Wander or ArConnect');
              return;
            }

            try {
              // Connect to wallet
              await window.arweaveWallet.connect(['ACCESS_ADDRESS', 'SIGN_TRANSACTION']);
              const addr = await window.arweaveWallet.getActiveAddress();
              setWallet(window.arweaveWallet);
              setAddress(addr);
            } catch (error) {
              console.error('Failed to connect wallet:', error);
            }
          };

          return { wallet, address, connectWallet };
        }
        ```
      </Step>
      <Step>
        ### Create file upload component with wallet

        ```jsx
        import React, { useState } from 'react';
        import { TurboFactory } from '@ardrive/turbo-sdk/web';

        function FileUploader() {
          const { wallet, address, connectWallet } = useWallet();
          const [uploading, setUploading] = useState(false);
          const [result, setResult] = useState(null);

          const handleFileUpload = async (event) => {
            const file = event.target.files[0];
            if (!file || !wallet) return;

            setUploading(true);
            try {
              // Create Turbo client using wallet signer
              const turbo = TurboFactory.authenticated({
                signer: wallet
              });

              const uploadResult = await turbo.uploadFile({
                fileStreamFactory: () => file.stream(),
                fileSizeFactory: () => file.size,
                signal: AbortSignal.timeout(60_000),
              });

              setResult(uploadResult);
              console.log('Upload complete:', uploadResult);
            } catch (error) {
              console.error('Upload failed:', error);
            } finally {
              setUploading(false);
            }
          };

          return (
            <div>
              {!wallet ? (
                <button onClick={connectWallet}>Connect Wallet</button>
              ) : (
                <>
                  <p>Connected: {address}</p>
                  <input
                    type="file"
                    onChange={handleFileUpload}
                    disabled={uploading}
                  />
                  {uploading && <p>Uploading...</p>}
                  {result && (
                    <div>
                      <p>Upload successful!</p>
                      <p>Transaction ID: {result.id}</p>
                      <a 
                        href={`https://arweave.net/${result.id}`} 
                        target="_blank" 
                        rel="noopener noreferrer"
                      >
                        View on Arweave
                      </a>
                    </div>
                  )}
                </>
              )}
            </div>
          );
        }

        export default FileUploader;
        ```
      </Step>
    </Steps>

    <Steps>
      <Step>
        ### Import from CDN

        Use ES module imports directly in your script tags. The bundled SDK includes all necessary polyfills.

        ```html
        <script type="module">
          import { TurboFactory } from 'https://unpkg.com/@ardrive/turbo-sdk@latest';
          
          // Your code here
        </script>
        ```
      </Step>
      <Step>
        ### Create minimal upload page

        ```html
        <!DOCTYPE html>
        <html lang="en">
        <head>
          <meta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>Turbo Upload</title>
        </head>
        <body>
          <h1>Upload to Arweave</h1>
          
          <input type="file" id="fileInput" />
          <button id="uploadBtn" disabled>Upload</button>
          <div id="status"></div>
          
          <script type="module">
            import { TurboFactory } from 'https://unpkg.com/@ardrive/turbo-sdk@latest';
            
            const fileInput = document.getElementById('fileInput');
            const uploadBtn = document.getElementById('uploadBtn');
            const statusDiv = document.getElementById('status');
            
            fileInput.addEventListener('change', (e) => {
              uploadBtn.disabled = !e.target.files[0];
            });
            
            uploadBtn.addEventListener('click', async () => {
              const file = fileInput.files[0];
              if (!file) return;
              
              statusDiv.textContent = 'Uploading...';
              
              try {
                const turbo = TurboFactory.authenticated({
                  privateKey: 'your-private-key' // Never do this in production!
                });
                
                const result = await turbo.uploadFile({
                  fileStreamFactory: () => file.stream(),
                  fileSizeFactory: () => file.size,
                });
                
                statusDiv.innerHTML = `Success! TX: <a href="https://arweave.net/${result.id}" target="_blank">${result.id}</a>`;
              } catch (error) {
                statusDiv.textContent = `Error: ${error.message}`;
              }
            });
          </script>
        </body>
        </html>
        ```
      </Step>
      <Step>
        ### Use wallet connectors (recommended)

        <Callout type="info">
          For production applications, we recommend using Wander for seamless wallet connection and transaction signing.
        </Callout>

        Wander provides a simple and secure way to connect Arweave wallets to your application. It supports multiple wallet providers and handles all the complexity of wallet integration.

        Learn how to integrate Wander for wallet connections: [Wander Connect Documentation](https://docs.wander.app/wander-connect/intro)

        ```javascript
        // Example: Using Turbo with Wander wallet connection
        import { TurboFactory } from 'https://unpkg.com/@ardrive/turbo-sdk@latest';
        
        // After connecting with Wander (see their docs)
        const turboClient = TurboFactory.authenticated({
          signer: window.arweaveWallet // Wander provides this
        });
        
        // Now you can upload files securely without exposing private keys
        const result = await turboClient.uploadFile({
          fileStreamFactory: () => file.stream(),
          fileSizeFactory: () => file.size,
        });
        ```

        Key benefits of using Wander:
        - No private key exposure in your code
        - Support for multiple wallet providers (ArConnect, Wander Wallet, etc.)
        - Simple integration with just a few lines of code
        - Built-in security features and best practices
      </Step>
